# Prueba_Desarrollo_Holanda

Prueba de empresa de Holanda de toma de turnos de motocicletas en tiempo real con javaScript y nodeJs.

La aplicaciÃ³n despliega cajas (los botones verdes) que representan 8 motociclistas que la empresa tiene en turnos cada 30 minutos.
